IoT Security Analysis using Network Monitoring and Python Automation

Group Members:Kavyashree and Amrutha

Objective

The objective of this project is to understand the fundamentals of IoT (Internet of Things) security by monitoring and analyzing network traffic generated by IoT devices. The project focuses on identifying insecure communication, unusual traffic patterns, and potential vulnerabilities. Python scripts are used to automate detection and analysis, helping security analysts secure IoT environments without directly interfering with device operations.

Problem Statement

IoT devices such as smart cameras, sensors, and home automation systems are widely used but often lack strong security mechanisms. Many devices communicate over networks without proper encryption, making them vulnerable to attacks like data interception, unauthorized access, and botnet infections.

Manual monitoring of IoT traffic is complex and inefficient. Security analysts require tools that can automatically capture, analyze, and detect suspicious behavior in IoT networks. This project demonstrates how network monitoring tools combined with Python automation can help identify vulnerabilities and improve IoT security.

Project Description

This project demonstrates IoT security analysis using:

Wireshark for manual packet inspection
Nmap for discovering devices and scanning ports
Python automation script to analyze captured network traffic

The system captures packets from IoT devices connected to the same network and analyzes them to identify:

IP addresses of devices
Communication protocols used
Suspicious or insecure data transmission
Repeated or abnormal traffic patterns
Features (Safe Version)
Uses only local network traffic (no external attack simulation)
Captures packets safely without modifying device behavior
Detects HTTP traffic (insecure communication)
Identifies unknown devices using IP and MAC addresses
Generates structured reports (JSON or text format)
Includes error handling and user-friendly output
No harmful interaction with IoT devices
